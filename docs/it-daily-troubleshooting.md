## 办公无线网（Wi-Fi）用户连接指南

本指南适用于公司办公无线网络 **T-Network** 接入流程。

所有设备首次接入前，必须将设备 MAC 地址提交 IT 加入白名单。

---

# 一、MacOS 系统

## 1️⃣ 获取设备 Wi-Fi MAC 地址

### macOS Ventura 及更高版本：

1. 打开【系统设置】
2. 进入【网络】→【Wi-Fi】
3. 找到 **T-Network**
4. 点击右侧三个圆点按钮
5. 将「私有 Wi-Fi 地址」设置为 **关闭**
6. 复制下方显示的 Wi-Fi 地址（MAC地址）

---

### macOS Ventura 以下版本：

1. 打开【系统偏好设置】
2. 进入【网络】→【Wi-Fi】
3. 点击右下角【高级】
4. 复制 Wi-Fi MAC 地址

---

## 2️⃣ 将 MAC 地址发送给 IT

请附带以下信息：

- 设备类型（MacBook / iMac 等）
- 使用人姓名
- 所在办公室

IT 将设备加入白名单。

---

## 3️⃣ 连接 Wi-Fi

1. 点击 **T-Network**
2. 输入 Wi-Fi 账号密码
3. 连接成功

---

# 二、Windows 系统

## 1️⃣ 获取 MAC 地址

1. 打开【开始菜单】
2. 进入【设置】
3. 点击【网络和 Internet】
4. 选择【WLAN】
5. 点击【硬件属性】
6. 复制“物理地址（MAC）”

发送给 IT。

---

## 2️⃣ 连接 Wi-Fi

1. 在 WLAN 可用网络中找到 **T-Network**
2. 点击连接
3. 输入账号密码

---

# 三、iOS 系统（iPhone / iPad）

## 1️⃣ 获取 MAC 地址

1. 打开【设置】
2. 点击【无线局域网】
3. 点击 **T-Network**
4. 将「私有无线局域网地址」设置为 **关闭**
5. 复制关闭后的无线局域网地址
6. 发送给 IT

---

## 2️⃣ 连接 Wi-Fi

输入 Wi-Fi 账号密码即可。

---

# 📌 注意事项

- 所有设备必须关闭“私有地址”功能
- 否则每次连接都会变更 MAC，导致无法上网
- 账号不得共享
- 如无法连接，请联系 IT
